# CampusConnect Demo Script

## 2-minute version

### Opening

"CampusConnect helps organizations run ambassador programs while also assessing technical potential from GitHub in under two minutes. Instead of keeping GitHub and ambassador work separate, we combine them into one decision flow."

### Step 1: Organization login

Use:

- `org@demo.com`
- `demo123`

Say:

"This is the organization dashboard. It shows operational health like reviews, GitHub adoption, and current shortlist activity."

### Step 2: GitHub Lab

Open `GitHub Lab`.

Say:

"With one GitHub username, the app generates a quick technical signal summary, including score, verdict, strengths, risks, and repository proof points."

If internet is available, analyze a public profile. If not, open a cached linked profile from the roster.

### Step 3: Ambassadors page

Open `Ambassadors`.

Say:

"Each ambassador is no longer just a points total. We now show GitHub score, Ambassador Fit Score, and pipeline status in one view."

### Step 4: Shortlist Board

Open `Shortlist Board`.

Say:

"This is where signals become decisions. Organizations can move ambassadors between Watch, Shortlist, and High Potential based on both execution and technical signal."

### Step 5: Ambassador login

Use:

- `arjun@demo.com`
- `demo123`

Say:

"From the ambassador side, users can submit tasks, track progress, and improve their public technical profile by linking GitHub."

### Close

"The core idea is simple: reduce evaluator effort, improve signal quality, and turn ambassador workflows into a better talent discovery system."

---

## Talking points for judging criteria

### Impact

- GitHub evaluation happens quickly with one username
- organizations get usable public-signal output immediately

### Innovation

- combines community workflow with technical profile intelligence
- Fit Score and Shortlist Board turn analysis into action

### Technical execution

- modular Streamlit app
- SQLite-backed workflow
- seeded demo data and stable demo accounts

### User experience

- dark themed dashboards
- clear role-based flows
- visual signals, not just tables and forms

### Presentation

- easy demo path
- immediate visible value
- supports both recruiter and ambassador stories
